# 智能备忘录 - 自动分类与行程规划

一个具有自动分类、记忆功能和智能行程规划的备忘录 Web 应用。

## 项目位置

```
/Users/linshi/memo-app/index.html
```

## 如何使用

### 方法一：直接打开

在终端执行以下命令：

```bash
open /Users/linshi/memo-app/index.html
```

### 方法二：拖入浏览器

用 Finder 打开 `/Users/linshi/memo-app/` 目录，将 `index.html` 文件拖入 Chrome/Safari/Edge 等浏览器。

## 功能介绍

### 1. 自动分类
输入备忘内容后，系统会根据关键词自动识别分类：
- **工作**：会议、项目、报告等
- **生活**：吃饭、购物、出行等
- **学习**：课程、考试、作业等
- **健康**：运动、健身、饮食等
- **购物**：下单、快递、优惠等
- **想法**：灵感、创意、思考等
- **其他**：未匹配的内容

### 2. 记忆存储
- 数据自动保存到浏览器 localStorage
- 关闭浏览器后数据不丢失
- 支持搜索历史备忘录

### 3. 智能行程规划 (新功能)

系统会自动从备忘录中提取时间信息，生成每日行程安排：

#### 时间识别支持
- **具体时间**："上午9点开会"、"下午2:30培训"
- **时间段**："早晨运动"、"晚上学习"
- **相对日期**："明天下午3点"、"下周一"
- **周几**："周三下午"、"周五晚上"

#### 优先级识别
- **紧急**：包含"紧急"、"重要"、"尽快"等关键词
- **一般**：普通任务
- **低优先级**：包含"有空"、"稍后"等关键词

#### 使用方法
1. 添加包含时间信息的备忘录
2. 点击"生成行程"按钮
3. 查看按时间排序的行程安排
4. 获取智能建议

#### 智能建议示例
- 工作任务较多时，建议休息时间
- 缺少运动安排时，建议运动时间
- 学习任务的最佳时间段建议
- 时间冲突提醒

### 4. 其他功能
- 搜索备忘录
- 按分类筛选
- 编辑和删除
- 统计数据（总数、今日新增、分类数）

## 使用示例

### 添加备忘录示例
```
明天上午9点开会讨论项目进度
下午2点培训新员工
晚上7点健身
周末购物买生活用品
```

### 生成的行程
```
09:00 ─── 开会讨论项目进度 [工作-紧急]
14:00 ─── 培训新员工 [工作]
19:00 ─── 健身 [健康]
```

### 智能建议
```
- 今天工作任务较多，建议在10:00-10:15和15:00-15:15安排休息时间
- 有紧急任务，建议优先完成
```
