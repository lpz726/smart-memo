# 智能备忘录 - 自动分类与行程规划

一个具有自动分类、记忆功能和智能行程规划的备忘录 Web 应用。

## 如何使用

下载之后

### 方法一：直接打开

在终端执行以下命令：

```bash
项目位置/index.html
```

### 方法二：将项目拖入浏览器

用 Finder 打开 `项目位置` 目录，将 `index.html` 文件拖入 Chrome/Safari/Edge 等浏览器。

## 功能介绍

### 1. 自动分类
输入备忘内容后，系统会根据关键词自动识别分类：
- **工作**：会议、项目、报告等
- **生活**：吃饭、购物、出行等
- **学习**：课程、考试、作业等
- **健康**：运动、健身、饮食等
- **购物**：下单、快递、优惠等
- **想法**：灵感、创意、思考等
- **其他**：未匹配的内容

### 2. 记忆存储
- 数据自动保存到浏览器 localStorage
- 关闭浏览器后数据不丢失
- 支持搜索历史备忘录

### 3. 语音控制 (新功能)

支持语音输入和语音命令控制，解放双手：

#### 语音输入
- 点击麦克风按钮开始语音输入
- 语音内容自动填入输入框并显示 AI 分类建议

#### 语音命令
| 命令 | 功能 |
|------|------|
| `添加 [内容]` | 直接添加备忘录 |
| `搜索 [关键词]` | 搜索备忘录 |
| `删除最新` | 删除最新一条备忘录 |
| `删除全部` | 清空所有备忘录 |
| `工作/生活/学习...` | 切换分类筛选 |
| `生成行程` | 生成每日行程 |
| `明天行程` | 查看明天的行程 |
| `清除搜索` | 重置搜索条件 |

#### 使用要求
- Chrome 或 Edge 浏览器
- 需要允许麦克风权限
- 需要网络连接

#### 快捷键
- `Ctrl+Enter` 或 `Cmd+Enter`（Mac）：快速添加备忘录

### 4. 智能行程规划

系统会自动从备忘录中提取时间信息，生成每日行程安排：

#### 时间识别支持
- **具体时间**："上午9点开会"、"下午2:30培训"
- **时间段**："早晨运动"、"晚上学习"
- **相对日期**："明天下午3点"、"下周一"
- **周几**："周三下午"、"周五晚上"

#### 优先级识别
- **紧急**：包含"紧急"、"重要"、"尽快"等关键词
- **一般**：普通任务
- **低优先级**：包含"有空"、"稍后"等关键词

#### 使用方法
1. 添加包含时间信息的备忘录
2. 点击"生成行程"按钮
3. 查看按时间排序的行程安排
4. 获取智能建议

#### 智能建议示例
- 工作任务较多时，建议休息时间
- 缺少运动安排时，建议运动时间
- 学习任务的最佳时间段建议
- 时间冲突提醒

#### 行程语音播报 (新功能)
点击"播报行程"按钮，系统会自动语音播报一天的行程安排：

**播报内容包括：**
- 开场问候与日期
- 行程总数概览
- 逐条播报：时间、分类、内容、优先级
- 智能温馨提示
- 结束语

**播报控制：**
| 按钮 | 功能 |
|------|------|
| ▶️/⏸️ | 播放/暂停 |
| ⏹️ | 停止播报 |
| ⏮️ | 上一条 |
| ⏭️ | 下一条 |

**可调节设置：**
- 语速：慢速/正常/快速/极速
- 音量：低/中/高

### 6. 其他功能
- 搜索备忘录
- 按分类筛选
- 编辑和删除
- 统计数据（总数、今日新增、分类数）

## 使用示例

### 添加备忘录示例
```
明天上午9点开会讨论项目进度
下午2点培训新员工
晚上7点健身
周末购物买生活用品
```

### 生成的行程
```
09:00 ─── 开会讨论项目进度 [工作-紧急]
14:00 ─── 培训新员工 [工作]
19:00 ─── 健身 [健康]
```

### 智能建议
```
- 今天工作任务较多，建议在10:00-10:15和15:00-15:15安排休息时间
- 有紧急任务，建议优先完成
```

## 技术栈

- HTML5
- CSS3 (渐变、动画、响应式)
- JavaScript (ES6+)
- localStorage (本地存储)
- Font Awesome (图标)
- Web Speech API (语音识别与合成)
